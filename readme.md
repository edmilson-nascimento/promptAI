# Promtps AI
-

## Principios basicos
Algumas perguntas para responder e assim criar um bom prompt:
- Quem?
- Quando?
- Para que?

Opcionais:
- Faça perguntas
- Tipo e saída de resposta

### Quem? 
Defina quem é voce para que a IA saiba o ponto de vista que deve adotar.
Exemplo: "Atue como um consultor/arquiteto SAP ABAP experiente e objetivo..."

### Quando?
Defina o contexto temporal para que a IA possa adaptar a resposta. 
Exemplo: "Um desenvolvimento que ja foi feito e preciso revisar..."

### Para que?
Defina o objetivo da sua pergunta para que a IA possa focar na resposta.
Exemplo: "Preciso Revisar uma classe de acordo com as guidelines..."

### Faça perguntas
Incluir perguntas na sua solicitação pode ajudar a IA a entender melhor o que você precisa.
Exemplo: "Me faça perguntas para contextualizar e apanhar mais informações caso seja preciso..."

### Tipo e saída de resposta
Para obter a informação no formato desejado, especifique o tipo de resposta que você espera.
Exemplo: "Forneça a resposta em formato de lista numerada com passos claros..."
